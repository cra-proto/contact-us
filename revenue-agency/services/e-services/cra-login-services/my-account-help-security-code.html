Get a security code by registering

You will be issued a security code as you register for My Account for Individuals, My Business Account, the MyCRA app, MyBenefits CRA app, or the CRA BizApp.

Sign in or register for an account

Update your mailing address

The security code will be mailed to the address we have on file. If you need to update your address without access to an account, call to update your mailing address.

Get a security code letter in an alternative format

Multiple formats are available for persons with a visual impairment, including Braille, e-text (CD), MP3, or large print.

Call 1-800-959-8281 to request a CRA security code in an alternative format.
